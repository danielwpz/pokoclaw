# Planned Builtin Skills

This directory is intentionally not scanned by the runtime skill loader.

Rules:
- Do not add `SKILL.md` files here.
- Use this directory to keep placeholder files for builtin/internal skills that are planned but not implemented yet.
- When a planned skill starts real implementation, move it out of `_planned/` into a normal builtin skill directory under `skills/<skill-name>/`.
